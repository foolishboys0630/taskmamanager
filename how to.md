&#x20; # ① Worker（Python）

&#x20; start-worker.bat



&#x20; # ② Backend（Node.js）

&#x20; start-backend.bat



&#x20; # ③ Frontend（Vite）

&#x20; start-frontend.bat



&#x20; ブラウザで http://localhost:5173 を開いてください。

### &#x20;**Workerとは？**



&#x20; 音声処理専用のPythonサーバーです。



&#x20; フロントエンド・バックエンドはJavaScript製ですが、音声処理（Whisper文字起こし・TTS音声生成・ff

&#x20; mpeg処理）はPythonライブラリを使うため、別プロセスとして分離しています。



&#x20; ブラウザ → Frontend(Vite) → Backend(Node.js) → Worker(Python) → ffmpeg/Whisper/TTS



&#x20; ---

&#x20; 起動に必要な事前インストール



&#x20; 1. Python（すでに入っています）



&#x20; 確認済み：Python 3.13.5 がインストール済み ✓



&#x20; 2. ffmpeg（要インストール）



&#x20; PowerShellを管理者として実行して：

&#x20; winget install ffmpeg

&#x20; インストール後、PowerShellを再起動して ffmpeg -version で確認。



&#x20; 3. Workerの起動



&#x20; cd "C:\\Users\\haruk\\OneDrive\\デスクトップ\\todo-project\\日本ローカルTTSアプリ\\worker"



&#x20; # 仮想環境を作成（初回のみ）

&#x20; python -m venv .venv



&#x20; # 仮想環境を有効化

&#x20; .venv\\Scripts\\activate



&#x20; # ライブラリをインストール（初回のみ・数分かかります）

&#x20; pip install -r requirements.txt



&#x20; # 起動

&#x20; uvicorn src.main:app --reload --port 8000



&#x20; http://localhost:8000 が起動したらOK。
