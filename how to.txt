  ---
  起動手順

  ターミナルを3つ開いて、それぞれ実行:

  # ① Worker（Python）
  start-worker.bat

  # ② Backend（Node.js）
  start-backend.bat

  # ③ Frontend（Vite）
  start-frontend.bat

  ブラウザで http://localhost:5173 を開いてください。



  ---
  使い始め前に必要なこと

  1. ffmpeg をインストール → winget install ffmpeg
  2. 設定画面 で Qwen API Key と Worker URL (http://localhost:8000) を設定
  3. 設定画面の「接続テスト」でWorker疎通確認

起動するプロセスは全部で3つ
